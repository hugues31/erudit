Erudit
======

![Erudit logo](https://github.com/hugues31/erudit/blob/main/logo.png?raw=true)

Erudit is a Reinforcement Learning framework written in Python.

This is a learning project. Not production ready.

You can learn more on how to use this library by taking a look at the [examples](https://github.com/hugues31/erudit/tree/main/examples).
